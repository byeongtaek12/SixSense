# 2. Key Features (주요 기능)

- **팀원 소개 홈 페이지**
    - 팀원들의 프로필을 카드 형식으로 한눈에 볼 수 있는 메인 화면 구성
- **자기 소개 페이지**
    - 각 팀원의 개인 정보를 상세히 확인할 수 있는 상세 페이지 구현
- **댓글 기능**
    - 팀원 소개 페이지에 댓글을 달아 소통할 수 있도록 구현
